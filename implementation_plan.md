# מעבר לאוטומציה מלאה בענן (Cloud Automation) ☁️

לבקשתך, אנחנו מעבירים את ה"מוח" והפעילות של הסוכן מהמחשב המקומי שלך - ישירות לענן.
זה אומר שהמחשב שלך יכול להיות כבוי, ואתה יכול להיות בחוף הים - והמערכת תמשיך לכתוב מאמרים ולפרסם פוסטים פעמיים בשבוע!

## איך נעשה את זה? (הארכיטקטורה)
מכיוון שהאתר שלך כבר מאוחסן ב-GitHub, נשתמש בשירות החינמי שלהם שנקרא **GitHub Actions** בתור ה"שרת" שלנו בענן.

1. **תזמון בענן:** ניצור קובץ תצורה שייכנס לפעולה כל יום שני וחמישי בשעה 19:00 (שעון ישראל).
2. **המוח (AI) בענן:** כדי שהענן יידע "לכתוב" מאמרים חדשים לבד, נכתוב סקריפט פייתון חדש (`auto_writer.py`) שיתחבר ל-API של מודל שפה (כמו ChatGPT של OpenAI או Claude), יבקש ממנו לכתוב מאמר שיווקי, וישתול אותו ישירות בקוד של `articles.html`.
3. **פרסום רשתות חברתיות:** באותו זמן, השרת יריץ גם את הקובץ `publish_social.py` שכבר יצרנו, ויפרסם פוסט לפייסבוק.
4. **שמירה:** השרת יעשה בעצמו `git commit` ו-`git push` לעדכון האתר באוויר.

> [!IMPORTANT]
> **אבטחת מידע (Secrets):**
> אי אפשר לשים סיסמאות בקוד גלוי. לכן תצטרך להיכנס להגדרות ה-Repository שלך ב-GitHub (תחת Settings -> Secrets and variables -> Actions) ולהזין לשם בצורה מאובטחת את ה-Token של פייסבוק, ובהמשך גם את ה-API Key של ה-AI.

---

## שאלות פתוחות לאישורך:
כדי שהשרת בענן יוכל "לחשוב" ולכתוב טקסטים יצירתיים (הרי אני - הסוכן שכרגע מדבר איתך - נשאר על המחשב המקומי שלך), הוא חייב גישה לשירות בינה מלאכותית. 

**איזה מודל תרצה לחבר לענן?**
1. **OpenAI (ChatGPT)** - דורש ממך לפתוח חשבון ב-platform.openai.com ולהוציא API Key (עולה גרושים, לפי שימוש).
2. **Gemini של Google** - יש להם מסלול חינמי למפתחים שאפשר להוציא ממנו מפתח.

ברגע שתבחר איזה מודל מועדף עליך, אני אכתוב את הסקריפט החדש לענן, אכין את קובץ ה-GitHub Actions, ואדריך אותך איך להכניס את המפתחות לאתר של GitHub! 🚀
